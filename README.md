# K-Cat
K-Cat transform your cat litter in a smart health and activity monitoring device, accessible from your smartphone!

## Use cases :
- Avoir statistiques sur la fréquence passage au toilette (détection passage du chat à l’entrée avec capteur magnétique sur la porte + balance pour vérifier si vraiment pipi ou caca)
- Avoir stats sur le poids du chat (encore balance)
- Etre notifié qu’il faut nettoyer  ou que c’est utilisé (paramétrable sur la fréquence) : sur l’appli + petite diode qui prévient
- Être notifié en cas de problème (changement poids / Changement habitude)
- Calibrer (renseigner les données dans l’appli ou première pesée -> système de pause pour pouvoir nettoyer la litière sans détection de changement de poids : l’appli demande confirmation qu’un nettoyage a été effectué)
- Générer un rapport (format pdf ou mail) pour envoyer au véto
- Système d’export des données

## Matériel :
- Détecteur ouverture de portes MC-38
- Balance à demander → “weight sensor OT 3243 ” *4 (il en faut un à chaque coin) + contrôleur “HX 711” aka “OT 3331” -> cellule de charge 5 kg sinon
- LED Super Flux série Orca R de Bivar
- Raspberry Pi
- Batterie 5V
- Piles rechargeables
- Fils
- Breadboard
